## Fish Shell
## [[Ubuntu]] or [[Debian Server|Debian]]
```Shell
sudo apt install fish
```

## [[Fedora]]/RHEL
```shell
sudo dnf install fish

or

sudo nano /etc/passwd
	edit /bash to /fish where appropriate
	logout and it will be fish

```

### To Set Fish as default shell
```shell
chsh -s /usr/bin/fish
```
